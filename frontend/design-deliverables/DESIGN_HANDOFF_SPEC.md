## 🎨 D-001: 設計系統理解筆記

### 設計哲學
**核心概念**: Editorial Elegance meets Playful Utility
- 將 AI 點餐體驗轉化為翻閱精緻美食雜誌的感受
- 溫暖的食物色系（奶油色、焦糖色、陶土紅）
- 非對稱排版 + 慷慨留白

### 關鍵設計決策

#### 1. 色彩系統
```
主色調：
- Cream #FFF8F0 (奶油白) - 主背景
- Caramel #D4A574 (焦糖色) - 主要按鈕、強調
- Terracotta #C85A54 (陶土紅) - 價格、重要資訊
- Sage #8B9D83 (鼠尾草綠) - 次要按鈕
- Charcoal #2D2D2D (炭黑) - 文字

功能色：
- Success #6B9D7F - 已確認、完成
- Warning #E89C5C - 超預算提示
- Error #C85A54 - 錯誤訊息
```

#### 2. 字體系統
```
Display Font: Cormorant Garamond (襯線體)
- 用於: H1, H2 主標題
- 特色: 優雅、雜誌感
- 大小: 3xl-5xl (39px-61px)

Body Font: Noto Sans TC (無襯線體)
- 用於: 內文、按鈕、卡片標題
- 特色: 清晰易讀、支援繁中
- 大小: base-2xl (16px-31px)

Handwriting Font: Caveat (手寫體)
- 用於: 推薦理由、註記
- 特色: 溫暖、人性化
- 大小: lg-xl (20px-25px)
- 特殊處理: -2deg 傾斜
```

#### 3. 間距系統
- 基準網格: 8px
- 常用間距: 4px, 8px, 16px, 24px, 32px, 48px
- 區塊間距: 96px (space-12)
- 卡片內距: 32px (space-6)

#### 4. 圓角系統
```
- Button: 16px (radius-button)
- Card: 24px (radius-card)
- Input: 12px (radius-input)
```

#### 5. 陰影系統
```
- Card Shadow: 0 4px 20px rgba(212, 165, 116, 0.15) (暖色調陰影)
- Floating: 0 12px 40px rgba(45, 45, 45, 0.25)
```

### 設計原則摘要

✅ **視覺層次清晰**
- 使用非對稱排版
- 慷慨的留白
- 字體對比（襯線標題 + 無襯線內文）

✅ **溫暖的食慾色系**
- 避免科技感藍紫漸層
- 擁抱食物的自然色調
- 所有色彩符合 WCAG AA 對比標準

✅ **流暢的互動回饋**
- 每個操作都有視覺反饋
- 動畫分三層次：功能性、回饋性、情感性
- 使用 Framer Motion 實作複雜動畫

### 關鍵組件設計要點

#### Button
- Primary: 漸層色 (caramel → terracotta)
- Secondary: Sage 綠色
- Outline: 邊框 + hover 變色
- Ghost: 透明背景 + hover 變色
- 狀態: default, hover, active, disabled

#### Card
- 預設: 白底 + card shadow
- Selected: 半透明 + success 色邊框 + ✓ 標記
- Hover: shadow 加深

#### DishCard (核心組件)
- 上方: 菜色照片佔位區（淡色背景 + emoji）
- 中間: 菜名（H3, Noto Sans TC bold） + 價格（Terracotta, large）
- 下方: 推薦理由（Caveat handwriting, -2deg 傾斜）
- 操作: 兩個按鈕（✅ 我要點這道、🔄 換一道）
